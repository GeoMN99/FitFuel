# FitFuel
### Author
George Ngunjiri

### Description
FitFuel is a pesronal fitness and nutrition tracking website that helps users stay on top of their daily health goals. Users can log workouts and meals and view a live dashboard that sows calries burned versus calories consumed in real time. It is built using HTML, CSS and JavaScript and uses the browser's built-in local storage to automatically save all entries.

### Technologies Used
- HTML
- CSS
- JavaScript

### Setup Instructions
1. Clone this repository
2. Open the project folder
3. Open 'index.html' in any browser
4. No installation or internet connection needed
